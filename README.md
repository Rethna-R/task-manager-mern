# MERN Task Manager

## Project Overview
This is a MERN Stack CRUD application for task management.
Users can add, edit, and delete tasks. Tasks are stored in MongoDB.

## Features
- Add a new task (Title + Description)
- Edit existing tasks
- Delete tasks
- Tasks displayed in a clean list
- Backend REST API using Node.js & Express
- MongoDB database storage

## Folder Structure
task-manager-mern/
├── backend/          # Node.js + Express backend code
├── frontend/         # React.js frontend code
├── screenshots/      # Screenshots for submission
└── README.md         # This file

## Technologies Used
- Frontend: React.js
- Backend: Node.js & Express.js
- Database: MongoDB & Mongoose
- Version Control: Git & GitHub

## Setup Instructions
1. Clone the repository:
   git clone https://github.com/Rethna-R/task-manager-mern.git
2. Backend setup:
   cd backend
   npm install
   npm start
3. Frontend setup:
   cd frontend
   npm install
   npm start
4. Open your browser at http://localhost:3000
